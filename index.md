Claude Code 작업 결과 + 결론 공유 인덱스.

매 보고서 = `<date>-<slug>.md` (front-matter 포함). GitHub Pages 자동 빌드 (Jekyll minima).

## 워크플로

```
1. 클로드 코드 작업
2. /share-report <markdown>
3. helper script — pandoc 또는 Jekyll 변환
4. git push → Pages 자동 빌드
5. URL 공유 (1 클릭)
```

발행 후 URL 패턴: `https://suoki87.github.io/reports/<year>/<month>/<day>/<slug>/`

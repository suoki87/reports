### 3-2. 학습으로 인한 5대 전략 변경

#### 변경 1 — 대법원 2021도15080 양면칼 발견 → 공격 논리 재구성

종전 내용증명 §4-가는 본 판례의 "**부검을 통하지 않고 사망의 의학적 원인을 정확하게 파악하는 데에는 한계가 있으므로 ··· 사망진단서의 기재가 객관적으로 진실에 반한다거나, 작성자가 그러한 사정을 인식하고 있었다고 함부로 단정하여서는 안 된다**"는 부분을 인용하고 있었다. 이는 **피고인(검안의) 유리한 법리**로서, 검안의 측 변호인이 본 내용증명을 받았을 때 즉시 자신의 방어 논리로 활용할 수 있는 부메랑이었다.

`korean-law:get_decision_text` 로 본 판례 원문 확보 후 양면칼 발견. 다음과 같이 재구성:

- 인용 제거: 부검 불요 한계 부분
- 인용 추가: **"환자의 진료 경과 종합 판단 의무" + "미필적 고의 = 외부 행위 형태로 추인"** 부분

| 시나리오 | 대응 |

| **지급** | 종결 |

### Stage 8. 검안의 형사 고발 (검안의 답변 부인 시)

## §5. 본 사건 핵심 카드 5종

본 사건 입증의 *결정적 객관 정황* 5종. 모든 외부 발송 공문서·신청서에 일관 인용.

### 카드 1 — 검안의 출입문 미진입 (시신 직접 관찰 X)

- **근거**: 아버지 이대순 + 친동생 유봉철 5/14 공증 진술서 §2(아버지)·§4(외삼촌)
- **객관 정황**: 출입문 바로 안쪽이 화장실 → 진입 시 시신 즉시 관찰 가능. 진입 X = 시신 미관찰 자동 입증
- **활용**: 의료법 §17 "직접 검안한 의사" 요건 미충족 + 미필적 고의 추인

### 카드 2 — 위치 불일치 ("거실에서 돌아가셨다" 답변)

- **근거**: 외삼촌 유봉철 5/14 공증 진술서 §6·결어 3
- **객관 정황**: 외삼촌이 부검 통보 무렵 누군가(경찰관·검안의 중 1인)에게 사망 정황 질문 → "거실에서 돌아가셨다" 답변 받음. 실제 위치 = 화장실 내
- **활용**: 답변자가 시신 직접 관찰 X 입증 + 경찰 보고서("화장실 문턱") vs 가족 답변("거실") 모순

### 카드 3 — 검안의 사후 항의 답변 3가지

- **근거**: 아버지 이대순 5/14 공증 진술서 §7·결어 5·6 (본인 직접 들음)
- **답변 내용**:
  1. "알코올 의존으로 사망하였다는 단정적 판단이 아니라 사인 옆에 '추정'이라고 기재한 바와 같이 의심된다는 정도의 의견"
  2. "엘리베이터에서 (옆집 친정 어머니가) 하시는 말씀 못 들으셨느냐. 술을 일곱 병이나 마신다고 하시지 않느냐"
  3. "보험 때문에 그러시죠. 그러면 보험은 보험사하고 싸우세요"
- **활용**: 검안의 본인이 사인 기재가 단정 X 자인 + 사인 기재 근거가 1회성 발언 1개 자인 + 미필적 고의 추인 자료

### 카드 4 — 사후 거주지 4가지 사실

- **근거**: 아버지 이대순 5/14 공증 진술서 §6·결어 4 (장례 종료 다음 날 본인 직접 확인)
- **객관 정황**:
  1. 화장실 외상 정황 (혈흔·청소 도구·변기 앞 충돌 시사)
  2. 부엌 김치찌개·반찬 정돈 (조리 직후 사망)
  3. 거주지 실내 정리정돈 (자기방임 양상 정면 배치)
  4. 술병 단 하나도 X (일곱 병 발언 정면 반박)
- **보강**: 과학수사대 촬영 사진 73장 정보공개 청구 확보 (IMG_0061~0145)

### 카드 5 — 故 이중순 회상 오인 (음주 답변 흐름)

- **근거**: 아버지 §3 + 외삼촌 §5
- **객관 정황**: 아버지가 본인 친동생 故 이중순(별개 사건, 평소 음주 빈도 높음) 회상 직후 경찰관 "망인도 술 마시느냐" 질문 → "마시긴 한다" 답변 = 故 이중순 회상 흐름의 일반 응답
- **활용**: 경찰 보고서 "유족 진술" 의 실제 의미 = 망인 일상 음주 인정 X 입증

## §6. 검안의 측 변호인 반박 가능 지점 8종 + 자기 방어

lawyer 2차 검수에서 식별된 검안의 측 변호인의 반박 가능 지점 8종 + 본 통지서·진술서의 자기 방어 강도 평가.

| # | 반박 지점 | 본 자료 자기 방어 강도 | 보강 권고 |
|---|---|---|---|
| 5-A | 사후 항의 답변 3가지 = 매형 1인 진술 (녹취 X) | 약함 | 통신사 통화 명세서 발급 + 첨부 |
| 5-B | 엘리베이터 일곱 병 발언자 신원 미확정 | 중간 | "발언 진위와 무관하게 사인 기재 근거 부족" 단락 추가됨 |
| 5-C | "거실에서 돌아가셨다" 답변자 모호 | 약함 → 중간 | 외삼촌 §6·결어 3 분리 강화 적용 완료 (F4) |
| 5-D | 공증 = 진정성 공증, 사실성 공증 X | 약함 → 중간 | 내용증명에 공증 법적 성격 한 줄 명시 적용 완료 |
| 5-E | 후두부 혈흔 의학적 해석 자격 | 중간 | 1차 출처 명시 적용 완료 + 정식 감정 절차 명시 |

## §12. 종합 평가 — 본 사건 승소 가능성

| 영역 | 평가 |
|---|---|
| **사인 정정 (외인사 인정)** | 객관 정황 4점 + 검안의 자백 3가지 + 미필적 고의 추인 법리 = **중상** |
| **사망보험금 회수 (동양생명 + 하나손해 상해 1억)** | KMDMC 외인사 인정 시 = **상** / 사인 미상 인정 시 = **중** |
| **검안의 형사 고발 유죄** | 대법원 2021도15080 미필적 고의 기준 엄격 = **중** |
| **검안의 행정 면허 징계** | 의료법 §17 위반 객관 정황 + 보건복지부 진정 = **중상** |
| **민사 손해배상** | 부수적 보강 카드 = **중** |
| **시효 압박** | 2029. 1. 30. 까지 충분 = **여유** |

**핵심 리스크**:
1. 검안의 측 변호인 즉시 선임 + 답변 3가지 부인 (5-A) — 통신사 명세서 발급으로 부분 보강
2. 보험사 자동 인지 → 사인 정정 전 처리 → 외인사 1억 X (premortem F3) — 어머니 통화 스크립트 통제
3. KMDMC 감정 5~9개월 — premortem F7 — 6/3 즉시 신청으로 차단
4. 의뢰인 진술인 진술 동요 — 5/14 공증인 인증서 + 객관 증거 4점 결합으로 보강

본 사건은 **객관 정황 4점 + 검안의 자백 3가지 + 미필적 고의 추인 법리** 의 3중 결합으로 사인 정정 + 보험금 회수 가능성이 매우 높은 상태이며, AI 도구·학습으로 인한 5대 전략 변경 (양면칼 발견·F1 비적용·진술서 격식 정밀화·내용증명 풀 메타·hook 셋업) 이 모두 적용되어 발송 전 점검 7건만 완료하면 5/19 발송 가능한 상태에 있다.

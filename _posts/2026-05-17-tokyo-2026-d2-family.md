### 18:00-19:30 · 트모로우랜드 테라스 = 저녁 + 일렉트리컬 명당 ⭐⭐

**핵심**: 식사 + 자리잡기 2-in-1 효율

- **18:00-18:30** **트모로우랜드 테라스** 도착 + 모바일 오더 결제 + 테라스석 확보
  - 모바일 오더 = 디즈니 앱 → 매장 선택 → 메뉴 → 결제 → 픽업 시간 슬롯 → 픽업 카운터에서 받기
  - **결제**: 신용·체크·프리페이드만 (현금 X). 신한 SOL 트래블 사전 등록 확정.
  - **5세 친화 메뉴**: 햄버거 세트·치킨·프렌치프라이·디저트
  - **임산부 메뉴**: 부드러운 빵·생야채 회피·익힌 메뉴
- **18:30-19:30** 테라스석에서 저녁 식사 + 매트 깔고 임산부 좌식 휴식
  - 일렉트리컬 코스 = 트모로우랜드 → 신데렐라성 앞 → 웨스턴랜드. 트모로우랜드 테라스 = 시작점 가까움 = **시야 확보 + 화장실 가까움 + 식사 통합**
  - 5세 = 벤치 좌석 OK = 끝까지 집중 가능

**왜 자유석 (DPA X)?**:
- DPA 일렉트리컬 = ¥2,500/인 × 3 = ¥7,500 (전용석·신데렐라성 전)
- 평일 (5/20 수) 자유석 = 30분-1시간 전 도착 충분. 트모로우랜드 테라스 = 평일 18시부터 자리잡기 가능
- 5세 시야 OK. 임산부 좌식 OK.
- **DPA 락 = 미녀와야수 1개만**. 일렉트리컬 자유석 = ¥0 절약
